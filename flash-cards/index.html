What is the difference between var, let and const?

var is function-scoped and hoisted. let and const are block-scoped. let can be updated, while const cannot be re-declared or updated.

What's the difference between `==` and `===`

`==` (loose equality) compares values after type coercion, while `===` (stricty equality) compares values and types without coercion

What is Hoisting?

The JavaScript engine moves declarations (but not initializations) to the top of their scope during compilation.

What is closure?

A function that retains access to variables from its outer scope even after the outer function has executed.

What is Promise?

An object representing an asynchronous operation's eventual result, which can be Pending, Fulfilled, or Rejected.
